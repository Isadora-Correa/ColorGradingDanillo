# Color Grading Danillo

Projeto React + Vite
## Rodar localmente

1. Instale as dependencias: `npm install`
2. Inicie o servidor: `npm run dev`

## Gerenciamento de Dados
O conteúdo do site (produtos, textos, etc.) é lido de arquivos `.json` na pasta `public/data`. Para fazer qualquer alteração no conteúdo, edite os arquivos JSON correspondentes diretamente no código-fonte.

A página `/admin` serve como um **visualizador** para conferir os dados, mas a edição é feita diretamente nos arquivos.


## Build

- `npm run build`
- `npm run preview`
