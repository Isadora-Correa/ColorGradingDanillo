# Color Grading Danillo

Projeto React + Vite rodando localmente, sem dependencias externas. Os dados do site ficam no `localStorage` do navegador e podem ser editados pela pagina `/Admin`.

## Rodar localmente

1. Instale as dependencias: `npm install`
2. Inicie o servidor: `npm run dev`

## Dados locais

O app usa o `localStorage` com a chave `nava-colorist-data-v1`. Para resetar os dados, limpe essa chave no navegador.

## Build

- `npm run build`
- `npm run preview`
